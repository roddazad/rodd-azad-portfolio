🌤️ Weather App

A dynamic weather app providing real-time weather updates using API integration.

🌍 Overview

The **Weather App** fetches real-time weather data from a third-party API, allowing users to check current weather conditions in any city. The app features a clean UI, search functionality, and dynamic updates.

✅ Features

  • ✔ Real-time weather updates using API integration
  • ✔ Search functionality for different cities
  • ✔ Dynamic background changes based on weather conditions
  • ✔ Temperature, humidity, and wind speed display
  • ✔ Responsive design for all devices

🚀 Future Enhancements

  • 🔹 Multi-day forecast display
  • 🔹 Dark mode option
  • 🔹 Favorite locations feature for quick access

🛠️ Technologies Used

HTML | CSS | JavaScript | Weather API

🔧 Dedicated Technology Breakdown

  • 📌 HTML5 - Structuring the weather interface.
  • 📌 CSS3 - Styling and improving UI/UX.
  • 📌 JavaScript - Handling API requests and DOM updates.
  • 📌 Weather API - Providing real-time weather data.
