✅ To-Do List App

A simple task manager with local storage support.

📝 Overview

This is a simple **To-Do List App** built using **HTML, CSS, and JavaScript** with **local storage** support. It allows users to add, remove, and store tasks in their browser for easy task management.

✅ Features

  • ✔ Add tasks with a text input field
  • ✔ Delete tasks with a remove button
  • ✔ Persistent storage using local storage
  • ✔ Minimalist and responsive design

🚀 Future Enhancements

  • 🔹 Task prioritization with color-coded categories
  • 🔹 Due date feature for each task
  • 🔹 Dark mode toggle for better user experience

🛠️ Technologies Used

HTML | CSS | JavaScript

🔧 Dedicated Technology Breakdown

  • 📌 HTML5 - Structuring the application layout.
  • 📌 CSS3 - Styling and enhancing the UI/UX.
  • 📌 JavaScript - Handling task management and local storage.
  • 📌 Local Storage - Ensuring task persistence after page reloads.
