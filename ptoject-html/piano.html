🎹 Piano Wonderland - Bilingual Piano Lessons Website

A bilingual website providing professional piano lessons in both English and Persian (Farsi).

🎼 Overview

This project is a bilingual website for professional piano lessons, offering both English and Persian (Farsi) versions. The website introduces the instructor, Mohammad Erfanian, and provides details about available piano lessons and services.

✅ Features

  • ✔ Bilingual Support: English and Persian versions available
  • ✔ Responsive Design: Built with Bootstrap for mobile-friendly experience
  • ✔ Fixed-top navbar with easy navigation
  • ✔ Hero Section with full-screen background image
  • ✔ About Section providing details about the instructor and his philosophy
  • ✔ Services Section with an interactive carousel for lesson types

🚀 Future Enhancements

  • 🔹 Booking system for scheduling lessons online
  • 🔹 Video tutorial integration for students
  • 🔹 Testimonials section for student feedback

🛠️ Technologies Used

HTML | CSS | Bootstrap | JavaScript

🔧 Dedicated Technology Breakdown

  • 📌 HTML5 - Structuring the web pages.
  • 📌 CSS3 - Custom styling for an elegant layout.
  • 📌 Bootstrap - Ensuring a responsive and visually appealing design.
  • 📌 JavaScript - Enhancing user interactivity and carousel functionality.
