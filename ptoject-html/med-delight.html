🍽️ Mediterranean Restaurant Website

A visually stunning, fully responsive website capturing the Mediterranean dining experience.

🏛️ Overview

Welcome to the Mediterranean Restaurant website. This project is a simple, responsive website that showcases the restaurant's story, menu, specialties, and contact/reservation features.

✅ Features

  • ✔ Home Page with hero section, specialties carousel, and reservation/contact section
  • ✔ About Page with restaurant story, philosophy, and team details
  • ✔ Menu Page displaying appetizers, main courses, desserts, and beverages
  • ✔ Fixed-top navbar for seamless site navigation
  • ✔ Specialties section with an interactive carousel
  • ✔ Fully responsive design using Bootstrap

🚀 Future Enhancements

  • 🔹 Online reservation system with backend integration
  • 🔹 Interactive menu with filtering and pricing
  • 🔹 Customer testimonials section with real reviews

🛠️ Technologies Used

HTML | CSS | Bootstrap | JavaScript

🔧 Dedicated Technology Breakdown

  • 📌 HTML5 - Structuring the web pages.
  • 📌 CSS3 - Custom styling for a Mediterranean aesthetic.
  • 📌 Bootstrap - Ensuring a fully responsive layout.
  • 📌 JavaScript - Enhancing user interactivity.
