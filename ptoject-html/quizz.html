📝 Quiz App

An interactive multiple-choice quiz application with score tracking.

📚 Overview

The Quiz App is a simple multiple-choice quiz application built using **HTML, CSS, and JavaScript**. It presents users with a series of questions, tracks their score, and provides a seamless interactive experience.

✅ Features

  • ✔ Multiple-choice questions
  • ✔ Score tracking
  • ✔ "Next" button to navigate through questions
  • ✔ Interactive UI with instant feedback

🚀 Future Enhancements

  • 🔹 Timer for each question
  • 🔹 Randomized question order
  • 🔹 Leaderboard to track high scores

🛠️ Technologies Used

HTML | CSS | JavaScript

🔧 Dedicated Technology Breakdown

  • 📌 HTML5 - Structuring the quiz layout.
  • 📌 CSS3 - Styling and improving UI/UX.
  • 📌 JavaScript - Handling question logic, score tracking, and UI updates.
