📊💰 Expense Tracker App

An intuitive web-based application for tracking, categorizing, and managing personal expenses.

📌 Overview

The Expense Tracker App is a web-based tool designed to help users track their expenses effortlessly. Users can **add, edit, and delete expenses**, ensuring complete control over their financial records. The app leverages **Local Storage** for persistent data management, allowing users to access their records anytime.

✅ Features

  • ✔ Add new expenses with details (name, amount, category, and date)
  • ✔ Edit existing expenses
  • ✔ Delete expenses from the list
  • ✔ Data persistence using Local Storage

🚀 Future Enhancements

  • 🔹 Sorting Expenses by Name, Amount, Category, or Date
  • 🔹 Exporting Expenses to CSV or PDF
  • 🔹 Expense Limit Notifications when exceeding budget
  • 🔹 User Authentication & Backend Integration for secure account tracking

🛠️ Technologies Used

HTML | CSS | Bootstrap | JavaScript | Local Storage

🔧 Dedicated Technology Breakdown

  • 📌 HTML5 - Structuring the application layout.
  • 📌 CSS3 - Enhancing UI/UX styling.
  • 📌 Bootstrap - Ensuring a responsive and modern UI.
  • 📌 JavaScript - Managing user interactions and local storage functionality.
  • 📌 Local Storage - Providing persistent expense data storage.
